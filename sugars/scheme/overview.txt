# Scheme (R5RS) Desugaring #

Representation: concrete surface syntax
Authorship: user-defined
Metalanguage: Scheme
Desugaring Order: OI
Phase: compile-time
Expressiveness:
  Arguments: any sort (really, any *syntax*)
  Result: any sort
  Deconstruction: yes
Safety:
  Syntax Safe: partly (paren safe only)
  Hygienic: yes
  Scope Safe: no
  Type Safe: no


# Installation Notes #

0. Install DrRacket: https://racket-lang.org/download/
1. Open files in DrRacket and hit 'Run'
   (Note that they are in the 'r5rs' language.)
